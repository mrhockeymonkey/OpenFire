# Creating new maps

Current isometric assets are from kenney.nl and are of size 256px , 512px.

Tiled Map Settings should be

* Orientation: Isometric (Staggered)
* Width: -
* Height: -
* TileWidth: 256
* TileHeight: 149
* StaggerAxis: Y
* StaggerIndex: Odd
* TileLayoutFormat: CSV
* TileRenderOrder: RightDown
* BackgroundColour: -
